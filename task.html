1.What is difference between Div&Span?

Div:-

  1. It is Block element
  2. It's extremely common to see divs used to group related paragraphs, images, headings, and links together.
  3. div is division element and it is used to combine or wrap the multiple elements
  4. Div it is used to divide the content when we biulding large websites.

Span:-

  1. It is a Inline element
  2. It creates inline containers for styling specific portions of text or elements within a block-level container.
  3. Span tags are used on small segments of text, links, images, and other HTML elements that appear inline with the surrounding content.
  4. Span tag is used to specify the specific content.

2.What is the semantic elements in HTML?
  1. A semantic element clearly describes its meaning to both the browser and the developer.
  2. Many web sites contain HTML code like: To indicates Header,navigation,section,article,aside and footer.
  • Header element represents a container for introductory content or a set of navigational links.
  • Nav element is intended only for major blocks of navigation links.
  • A section is a thematic grouping of content, typically with a heading.
  • An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.
  • Aside element defines some content aside from the content it is placed in (like a sidebar).
  • Footer element defines a footer for a document or section.
